# 004 – Interactive Liquid Gradient (Study)

Este projeto é um **estudo avançado** sobre gradientes animados e interativos
utilizando **WebGL, Three.js e Shaders (GLSL)**.

## Objetivo do estudo

- Entender como funcionam gradientes líquidos animados
- Explorar renderização via GPU (WebGL)
- Analisar o uso de shaders fragmentados
- Estudar interação por mouse/touch em shaders


## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Three.js
- WebGL
- GLSL (Fragment Shader)
