# 021 — Drag and Drop Board

Quadro interativo com funcionalidade de arrastar e soltar cartões entre colunas.

## Conceitos praticados
- Drag and Drop API
- Manipulação do DOM
- Eventos (`dragstart`, `dragover`, `drop`)
- Organização visual estilo Kanban

## Tecnologias
- HTML
- CSS
- JavaScript puro
