# Event KeyCodes

Projeto interativo que mostra informações da tecla pressionada
em tempo real.

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)

## Conceitos praticados
- Eventos de teclado
- Manipulação do DOM
- Estilização moderna com Glassmorphism
- Interação em tempo real

## Objetivo
Aprender como capturar e exibir dados de eventos do teclado
de forma clara e visual.
