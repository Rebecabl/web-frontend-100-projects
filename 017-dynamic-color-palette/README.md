# Dynamic Color Palette Generator

Gerador de paletas de cores dinâmicas feito com HTML, CSS e JavaScript.

## Funcionalidades

- Geração aleatória de cores
- Exibição do código HEX
- Clique para copiar a cor
- Feedback visual ao copiar
- Layout moderno e responsivo

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Objetivo

Praticar manipulação de DOM, eventos, lógica e UX simples.
