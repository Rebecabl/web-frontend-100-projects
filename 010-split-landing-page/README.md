# Split Landing Page

Projeto de landing page dividida em duas áreas interativas, onde cada lado
ganha destaque ao passar o mouse.

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)

## Conceitos praticados
- Flexbox
- Transições CSS
- Interação com eventos de mouse
- UI moderna

## Objetivo
Treinar efeitos visuais e interação simples para páginas institucionais
e landing pages.
