# 023 — Kinetic Loader

Loader animado utilizando apenas CSS, com rotação e sobreposição de camadas.

## Funcionalidade
- Animação contínua de carregamento
- Camadas com delays para efeito cinético
- Design minimalista e elegante

## Conceitos praticados
- CSS Animations
- Keyframes
- Transform e timing
- Design de loaders modernos

## Tecnologias
- HTML
- CSS
