# Animated Navigation

Menu de navegação com animação de abertura e fechamento, desenvolvido com HTML, CSS e JavaScript puro.

## Funcionalidades
- Menu retrátil animado
- Abertura e fechamento via clique
- Layout simples e responsivo

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo do Projeto
Praticar manipulação de classes no JavaScript e animações com CSS.
