# 013 — Random Choice Picker

Projeto interativo que sorteia uma opção aleatória a partir de
uma lista digitada pelo usuário.

##  Funcionalidade
- Entrada de texto com múltiplas opções
- Criação dinâmica de tags
- Animação de seleção aleatória
- Destaque visual da escolha final

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)

##  Conceitos praticados
- Manipulação de strings
- Arrays e números aleatórios
- Manipulação do DOM
- Timers (`setInterval` e `setTimeout`)
- Eventos de teclado

##  Objetivo
Treinar lógica de programação e interação com o usuário
de forma visual e divertida.
