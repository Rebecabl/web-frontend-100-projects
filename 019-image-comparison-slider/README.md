# Projeto 018 – Custom Range Slider

Este projeto demonstra o uso de um **slider customizado** para controlar
o nível de desfoque de uma imagem em tempo real.

## Funcionalidade

- Slider controla o valor de blur
- Alteração visual imediata
- Interface limpa e simples
- Projeto focado em UX e interação

## Tecnologias

- HTML
- CSS
- JavaScript puro

## Objetivo de Estudo

- Trabalhar com input type="range"
- Manipular estilos dinamicamente com JavaScript
- Criar feedback visual em tempo real
