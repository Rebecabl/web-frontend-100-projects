# Sticky Navigation

Projeto de navegação fixa que reage ao scroll da página.

## Funcionalidades
- Header grande no topo
- Redução automática ao rolar a página
- Fundo com blur e sombra
- Navegação sempre acessível

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo
Demonstrar controle de scroll, manipulação de classes e boas práticas de UX.
