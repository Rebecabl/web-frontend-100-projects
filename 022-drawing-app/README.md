# 022 — Drawing App

Aplicação de desenho interativo utilizando o elemento Canvas.

## Funcionalidades
- Desenho livre com mouse
- Escolha de cor
- Controle de espessura do pincel
- Limpeza do canvas

## Conceitos praticados
- Canvas API
- Eventos de mouse
- Manipulação de estado
- Interface interativa

## Tecnologias
- HTML
- CSS
- JavaScript puro
